# Job Market Intelligence Dashboard

A market intelligence tool that pulls live remote job postings, identifies
data/analytics-related roles, tracks in-demand skills, and visualizes hiring
trends in an interactive Streamlit dashboard.

## Features
- Pulls real job data from the [RemoteOK](https://remoteok.com/api) API
- Filters for data/analytics-related roles
- Detects in-demand skills using keyword matching (Python, SQL, AWS, etc.)
- Visualizes top skills, top hiring companies, and top job titles
- Interactive filtering by location, work type, and skill
- Skill recommendation engine ("if you know X, also learn...")
- Downloadable filtered dataset (CSV)

## Project Structure
```
.
├── fetch_jobs.py     # Pulls and processes job data -> jobs.csv
├── app.py            # Streamlit dashboard
├── jobs.csv          # Cached dataset (generated by fetch_jobs.py)
├── requirements.txt
└── README.md
```

## Workflow
```
fetch_jobs.py -> jobs.csv -> app.py
```

## Setup & Run

1. **Clone the repo**
   ```bash
   git clone <your-repo-url>
   cd job-market-dashboard
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Fetch the latest job data**
   ```bash
   python fetch_jobs.py
   ```
   This creates/updates `jobs.csv`.

4. **Run the dashboard**
   ```bash
   streamlit run app.py
   ```

## Notes
- A pre-generated `jobs.csv` is included so the dashboard runs out of the
  box. Re-run `fetch_jobs.py` to refresh the data.
- Most RemoteOK postings don't include a salary, so the "Avg. Salary" metric
  may show "N/A" depending on the dataset.

## Use Case
Companies, recruiters, and job seekers can use this dashboard to understand
where hiring demand is moving — for workforce planning, recruiting strategy,
competitive intelligence, or training program design.
